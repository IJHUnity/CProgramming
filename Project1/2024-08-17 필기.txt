데이터 종류
int     정수
float  실수
char  문자

화면 출력 기능
printf("적고싶은 문장");

화면 출력(서식) -> 위에 있는 정수, 실수, 문자를 작성하는 방법
printf("%d", 10);  10이 출력됩니다.
printf("%f", 3.14f); 3.140000가 출력됩니다.
printf("%c", 'a'); 문자 a가 출력됩니다.
printf("%.2f", 3.14f); 3.14가 출력됩니다.(소수점 2자리까지)

변수 만드는 방법(프로그램에서 데이터 저장해서 표현하는 방법)
자료형 변수명;
ex) 정수형 변수 number를 만들겠다 --> int number;

변수에 값 넣는 방법
변수명 = 값;
ex) 위에서 만든 number에 5를 넣겟습니다. --> number = 5;

위의 두 작업은 동시에 진행할 수 있습니다.
ex) 문자형 변수 alpha에 'a'를 넣겠습니다. --> char alpha = 'a';

#else나 else if는 필요 없다면 사용하지 않아도 됩니다.( if문의 보조 문법 개념)

크기 비교에 사용되는 기호
a > b, a < b , a >= b, a <=b, a == b(a와 b는 서로 (크기가) 같다.), a != b(a와 b는 서로 (크기가) 다르다.)
